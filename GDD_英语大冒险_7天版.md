## 7+1 错题巩固日 (Day 8)

| 项目 | 数值 |
|------|------|
| 时长 | 40 分钟 |
| 题目来源 | Day 1~7 答错的题目 |
| 存储方式 | localStorage（浏览器本地存储） |
| 关卡 | 3 关: 词汇错题 → 语法错题 → 听力+Boss混编 |
| 重置机制 | Day 8 完成后可选择清空错题记录 |

### 实现思路

1. 每天游戏结束时，将答错的题目数据保存到 `localStorage.wrongAnswers`
2. 数据结构：
   ```json
   [{ day:1, stage:'vocab', question:{...}, wrongAnswer:'...', correctAnswer:'...' }, ...]
   ```
3. Day 8 页面读取 localStorage，按关卡分组出题
4. 答对的题从错题集中移除，答错的保留
5. 全部清空即通关

## 计分公式

```
单题得分 = 100 × 难度系数 × 速度系数 × 连击系数

难度系数: easy=1.0, medium=1.5, hard=2.0
速度系数: 1.0 ~ 1.3 (剩余时间比例)
连击系数: 1.0 ~ 1.5 (每连击+0.05, 上限10连)
```

## 设计要点

1. **全英文听力**：题目和选项均为英文，培养语感
2. **无跟读模块**：去除语音识别，降低手机兼容门槛
3. **手动推进**：每道题答完后显示反馈，点"下一题"才继续
4. **50%冗余**：每关题库比实际出题多50%，重复游玩有新鲜感
5. **Boss关限时**：每题15秒倒计时，训练快速反应
6. **连击激励**：连续答对分数递增，答错归零
