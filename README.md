# 英语大冒险 🗺️ English Adventure

沪教版（牛津上海版）5B 小学英语闯关游戏，一个**自包含 HTML 文件**，无需安装任何软件，用 Chrome 浏览器打开即可玩。

## 🏗️ 项目结构

```
├── Day1_What_a_mess.html     ← 第1天：What a mess!
├── Day2_Watch_it_grow.html   ← 第2天：Watch it grow!
├── Day3_Listen_Food.html     ← 第3天：Food & Drinks
├── Day4_Favourite_Things.html← 第4天：Favourite Things
├── Day5_Animals_Signs.html   ← 第5天：Animals & Signs
├── Day6_Weather_Changes.html ← 第6天：Weather Changes
├── Day7_Review.html          ← 第7天：复习总结
├── GDD_英语大冒险_7天版.md    ← 游戏设计文档
└── batch_build.py            ← 批量生成脚本
```

## 🎮 游戏特色

- **4 关 40 题/天**：词汇(17) → 语法(13) → 听力(5) → Boss(5)
- **7+1 天体系**：7天闯关 + 错题重做巩固
- **错题回顾机制**：全部题目完成后自动重做错题，直到全部答对
- **5秒倒计时**：每题答完后强制5秒回顾，加深印象
- **计分系统**：综合难度、速度、连击加成
- **完全离线可用**：一个 HTML 文件，Chrome 直接打开

## 👶 目标用户

小学五年级学生，沪教版英语 5B 下册同步练习
