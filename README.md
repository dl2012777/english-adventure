## 🎮 游戏特色（双学科通用）

| 关卡 | 英语版 | 数学版 |
|:----:|:------:|:------:|
| 📝 第1关 | 词汇闯关（17题·选择题） | 口算速递（17题·选择题） |
| 🧠 第2关 | 语法迷宫（13题·选择题） | 概念通关（13题·选择题） |
| 🎯 第3关 | 听力挑战（5题·听音选择） | 应用挑战（5题·纸上作答） |
| ⚡ 第4关 | Boss关（5题·限时30秒） | Boss关（5题·限时60秒·纸上作答） |

**共同机制：**
- ✅ 每题答完显示大勾/大叉 + 解题思路强化
- ✅ **5秒倒计时**后才能点下一题，强制回顾
- ✅ 全部答完后**错题重做**，直到全对
- ✅ 计分系统（速度+连击+难度加成）
- ✅ 应用挑战/Boss关 → **纸上做题 + 大人对照答案批改**
- ✅ 结果页支持「📋 查看题目清单」—— 所有题目+答案一目了然
- ✅ 手机浏览器直接打开即可使用
